# 05_Output

This is the only final deliverable area.

Only gate-approved, clean, reproducible deliverables belong here.

- `rtl/` - signed RTL code.
- `tb/` - signed directed testbench code.
- `uvm/` - signed UVM verification code.
- `fpga/` - signed FPGA package.
- `reports/` - final audit, trace, compliance, Loop1, Loop2, and Loop3 reports.

Do not place runtime logs, temporary files, or unreviewed intermediate artifacts here.

`manifest.yaml` records the canonical output contents and signoff evidence.
