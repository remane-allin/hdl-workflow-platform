# Project Memory

Project memory is grouped by pipeline node for fast recovery.

- `00_global/` - project brief, current state, decisions, open questions.
- `01_docparse/` - document parsing and normalization notes.
- `02_loop1/` - RTL/TB bring-up notes and Loop1 decisions.
- `03_loop2/` - UVM, coverage, and bug closure notes.
- `04_loop3/` - FPGA implementation and board validation notes.
